# fastapi-app-deployment

**IaC: деплой стека с Ansible, Registry и Vault**

Репозиторий с Ansible инфраструктурой для развертывания полного стэка (бэкенд, фронтенд, БД, Nginx) и сопутствующих компонентов на целевом сервере с использованием HashiCorp Vault секретов из приватного gitlab регистри.  


## Что делает

- Подготавливает хост
- Тянет образы из приватного gitlab registry
- Деплоит контейнеризированные сервисы (FastAPI бэкенд + Postgres + Vue фронтенд)
- Настраивает Nginx на хосте для работы с доменом из HashiCorp Vault
- Работает с секретами и сертификатами через HashiCorp Vault
