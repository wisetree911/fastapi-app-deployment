# fastapi-app-deployment

**IaC: деплой FastAPI-стека с Ansible, Docker и Vault**

Репозиторий с инфраструктурой и плейбуками для развертывания backend-сервиса на FastAPI и сопутствующих компонентов на целевых серверах с использованием Docker.  

---

## Что делает

- Подготавливает хост
- Деплоит контейнеризированные сервисы (FastAPI + зависимости)
- Тянет образы из приватного registry
- Применяет конфиги/шаблоны
- (Опционально) Работает с секретами через HashiCorp Vault
